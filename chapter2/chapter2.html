Chapter 2 基本概念

语法

注释同C++和Java
在函数的第一行写"use strict";告诉支持的JavaScript引擎切换到严格模式
每个语句最有要加;（同C++和Java）

变量

var

所有变量均由var来申明，具体类型由最后的赋值类型确定
初始化变量不会将var标记为初始化类型，但不建议修改变量类型赋值
在函数内声明变量不用var可以声明一个全局变量

可以用typeof运算符获得变量的类型
null是Object类型，Undefined类型只有一个值

整型

整形变量十进制直接赋值，八进制在数字前加上0，十六进制加上0x
进行计算时，所有数值最终都将转化成十进制
尽量不要测试特定的浮点数值
正无穷Infinity，负无穷-Infinity，可用isFinite(数)判断是否为无穷
NaN：Not a Number。用于表示一个本来要返回数值的操作数却未返回数值的情况（异常抛出）
string类型的数字可以转化为整型，所有isNaN()返回的值为false
将其他类型的非数值转化为数值的函数Number()、parseInt()、parseFloat()，前一个适用于所有，后两个适用于字符串
null对应0，undefined对应NaN，如果是对象则调用valueOf()

字符串

String.length中转义序列算一个字符

Object

var object=new Object();
bool hasOwnProperty(String propertyName)御用检查给定属性是否存在于对象实例中
bool isPrototypeOf(Object object)判断传入对象是否是另一个对象的原型
bool propertyIsEnumerable(String)判断传入属性是否可用于for-in语句

操作符

一元操作符

同C
自增自减，相反数，加减，按位非（~），按位与（&），按位或（|），按位异或（^），左移?位（<>），无符号右移（>>>）

布尔操作符

非（!），与（&&），或（||）

乘除求模加减，大小与等于，相等不相等，全等（===）还会判断类型

条件操作

var max=(num1>num2)?num1:num2;

语句

if-else，do-while，while，for，for-in

label语句

类似于goto的标记

break-continue
可以break/continue label;

with语句

将代码作用域设置到一个特定对象中
with(expression) statement;
statement中的变量如果找不到，就会到expression中去找

swtich-case
switch可以为任意变量，case可以为任意变量，甚至可以是条件判断

函数

function functionName(args...){statements;}
args不需要声明类型
alert(var)输出变量

函数的参数列表数量不重要，可以通过arguments[i]来使用第i个参数
参数传递都是值，不可能传引用
没有函数重载，后一个函数会覆盖前一个函数
