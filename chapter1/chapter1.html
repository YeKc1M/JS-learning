<script>元素

6个属性

async：可选。表示应该立即下载脚本，但不应妨碍页面中其他操作。只对外部脚本文件有效
charset：可选。表示通过src属性指定的代码的字符集。很少使用
defer：可选。表示脚本可以延迟到文档完全被解析和显示后再执行
language：废弃。
src：可选。表示包含要执行代码的外部文件
type：可选。language的替代属性，表示编写代码使用的脚本语言的内容类型，一般为text/javasript

延迟脚本：在script元素中加上defer="defer"
异步脚本：async bool属性

嵌入JavaScript代码

直接嵌入
<script type="text/javascript">
function sayHi(){
alert("Hi");
}
</script>
如果代码中包含</script>，使用<\/script>替代

引用外部js文件
script type="text/javascript" src="文件名.js"></script>
带有src属性的script元素不应该在标签之间再加JavaScript代码

所有script元素都放在head元素中，css文件引用后
在XHTML中，<会被当做新的标签，为保证代码可读性
<script type="text/javascript"><![CDATA[
代码
]]></script>
若不兼容CData，则将该部分注释掉不加CDATA

若浏览器不支持JavaScript，可在body中增加noscript元素区域显示不支持JavaScript时显示的内容
